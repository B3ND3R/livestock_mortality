# Livestock Mortality Rate Estimation 
This project is a collaboration between the World Food Programme Innovation Accelerator and the UC Berkeley [add program title] Capstone Program. 

The goal of this project is to estimate livestock mortality rates in developing countries using data from various sources, including government records, animal health databases, and social surveys. This repository is in development as of Spring 2026. 

## Setup instructions
Clone this repo and run `pixi install` in the root directory. 

## Data
The this project will use the data from [Kenya](https://hdl.handle.net/10568/131219) and [Nigeria](https://dataverse.harvard.edu/dataset.xhtml?persistentId=doi:10.7910/DVN/JIRVS9). The 01_notebook only uses the Kenya data.
